# Sudoku Solver Python

A Python-based Sudoku Solver that uses the Backtracking Algorithm to solve any valid Sudoku puzzle.

## Features

- Solve Sudoku automatically
- User-defined puzzle input
- Backtracking Algorithm
- Displays solved grid
- Handles invalid puzzles

## Technologies Used

- Python 3
- Backtracking Algorithm

## Installation

```bash
git clone https://github.com/yourusername/sudoku-solver-python.git
cd sudoku-solver-python
python sudoku_solver.py
```
